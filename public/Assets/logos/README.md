# CIPC Logo Variations

This folder contains different variations of the CIPC Paramedical Council logo for use across the website.

## Logo Files Required:

1. **logo-main.png** - Main full logo (800x800px) - For homepage, about page
2. **logo-header.png** - Header logo (200x60px) - For navigation header
3. **logo-footer.png** - Footer logo (150x150px) - For footer section
4. **logo-icon.png** - Icon only (64x64px) - For favicon, small spaces
5. **logo-square.png** - Square version (200x200px) - For dashboard, cards
6. **logo-white.png** - White version (200x200px) - For dark backgrounds
7. **logo-transparent.png** - Transparent background (200x200px) - For overlays
8. **logo-favicon.png** - Favicon (32x32px) - For browser tab
9. **logo-banner.png** - Banner version (800x200px) - For wide spaces
10. **logo-certificate.png** - Certificate version (400x400px) - For certificates

## Logo Specifications:

- **Format**: PNG with transparency where needed
- **Colors**: 
  - Yellow (#FFD700 or similar)
  - Blue (#003366 or similar)
  - Green (#228B22 or similar)
  - Red (#DC143C or similar)
- **Text**: "CIPC PARAMEDICAL COUNCIL" with Devanagari text on banner
- **Elements**: Caduceus symbol, open book, laurel wreaths, stars

## Usage Guidelines:

- Use logo-main.png for prominent displays
- Use logo-header.png in navigation bars
- Use logo-footer.png in footer sections
- Use logo-icon.png for small spaces and favicon
- Use logo-white.png on dark backgrounds
- Use logo-transparent.png when overlaying on images
